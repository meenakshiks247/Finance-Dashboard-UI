# Finance Dashboard UI

A beginner-friendly React + Vite starter for a finance dashboard internship assignment.

## Tech Stack
- React
- Vite
- Tailwind CSS

## Folder Structure
- `src/components` - reusable UI and layout pieces
- `src/pages` - screen-level components
- `src/data` - mock data and constants
- `src/context` - shared UI state
- `src/utils` - helper functions

## Dashboard Sections
- Top navigation bar
- Summary cards
- Chart section
- Insights section
- Transactions section

## Scripts
- `npm run dev` - start the development server
- `npm run build` - create a production build
- `npm run preview` - preview the production build

## Notes
- Functional components and React hooks only
- Responsive layout with a clean dashboard shell
- Easy to extend with real data and additional pages
